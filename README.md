# trainers-showcase

Отдельный мини-проект для запуска 5 тренажеров без iframe.

## Что делает

- Показывает одну страницу с 5 кнопками.
- Каждая кнопка открывает встроенный тест внутри проекта.
- Не содержит backend.
- Деплоится отдельно от основной платформы.

## Список кнопок

- Добавить часть слова (начало/конец) -> `/tests/test-02-main`
- Исправь букву в слове -> `/tests/test-13-main`
- Найти окончание слов -> `/tests/test-04-main`
- Составление фразы по картинке -> `/tests/test-01-mame`
- Фраза и картинка -> `/tests/test-14-main`

## Локальный запуск

```bash
cd trainers-showcase
npm install
npm run dev
```

Открыть: `http://localhost:3000`

## Деплой на VPS

1) Создать `.env`:

```bash
cp env.vps.example .env
```

2) Запустить деплой:

```bash
SSHPASS='твой_пароль_vps' ./scripts/deploy-vps-now.sh
```

Скрипт:

- удаляет старую папку `/opt/trainers-showcase`,
- загружает новую версию проекта,
- собирает и поднимает контейнер `trainers-showcase-web`.

После деплоя открыть:

- `http://170.168.16.142:3010`
